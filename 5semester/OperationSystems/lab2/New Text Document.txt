Описание: Команда ls используется для отображения списка файлов и директорий в текущей директории или в указанной директории.

Описание: Команда pwd (Print Working Directory) отображает полный путь к текущей рабочей директории.

Описание: Команда rmdir (Remove Directory) используется для удаления пустых директорий.

Описание: Команда touch используется для создания пустых файлов или обновления времени последнего доступа и модификации существующих файлов.

Описание: Команда cp используется для копирования файлов и директорий.

Описание: Команда mv используется для перемещения или переименования файлов и директорий.

Описание: Команда rm используется для удаления файлов и директорий

Описание: Команда su (substitute user) используется для смены пользователя в текущей сессии терминала.

Описание: Команда whoami отображает имя текущего пользователя, под которым вы вошли в систему.

Описание: Команда cat (concatenate) используется для отображения содержимого файлов, объединения нескольких файлов и создания новых файлов.

Описание: Команда less позволяет просматривать содержимое файлов постранично. Это особенно полезно для больших файлов.

Описание: Команда more также позволяет просматривать содержимое файлов постранично. Она похожа на less, но имеет меньше возможностей.

Описание: Команда head отображает первые несколько строк файла. По умолчанию отображает первые 10 строк.

Описание: Команда tail отображает последние несколько строк файла. По умолчанию отображает последние 10 строк.

Описание: Команда lpstat используется для отображения информации о состоянии принтеров и заданиях на печать. Она позволяет просматривать информацию о принтерах, очередях печати и статусе текущих заданий.

Описание: Команда lpr используется для отправки задания на печать в очередь печати. Она позволяет отправить один или несколько файлов для печати.

Описание: Команда lpq используется для отображения состояния очереди печати. Она показывает список всех заданий на печать в очереди, их номера и статус.

Описание: Команда lprm используется для удаления заданий из очереди печати. Она позволяет удалить одно или несколько заданий по их номерам.

Описание: Команда chgrp используется для изменения группы, к которой принадлежит файл или директория. Она позволяет установить новую группу для указанного файла или директории.

Описание: Команда chown используется для изменения владельца и/или группы владельца файла или директории. Она позволяет установить нового владельца и/или группу для указанного файла или директории.

Описание: Команда chmod используется для изменения прав доступа к файлам и директориям. Она позволяет установить, какие пользователи (владелец, группа, остальные) могут читать, записывать или выполнять файл.

Описание: Команда locate используется для быстрого поиска файлов и директорий на основе базы данных, которая обновляется периодически. Она обычно быстрее, чем другие методы поиска, потому что ищет в предсозданной базе данных, а не в реальной файловой системе.

Описание: Команда grep используется для поиска строк, содержащих определенные шаблоны, в файлах. Это мощный инструмент для фильтрации текстовых данных с использованием регулярных выражений.

Описание: Команда find используется для поиска файлов и директорий на основе различных критериев, таких как имя, размер, дата изменения и т. д. В отличие от locate, find ищет непосредственно в файловой системе.
