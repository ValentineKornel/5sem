1. Получите список всех существующих PDB в рамках экземпляра;
select NAME from v$pdbs;

2. Напишите запрос, позволяющий получить перечень экземпляров.
select INSTANCE_NAME from V$INSTANCE;

3. Получите перечень файлов всех журналов повтора инстанса.
SELECT * FROM V$LOGFILE;

4. Получите список управляющих файлов.
SELECT * FROM V$CONTROLFILE;

5.	Определите общий размер области SGA.
SELECT SUM(VALUE) FROM V$SGA;

6.	Определите объем доступной свободной памяти в SGA.
SELECT CURRENT_SIZE FROM V$SGA_DYNAMIC_FREE_MEMORY;

7.	Определите размеры пулов КЕЕP, DEFAULT и RECYCLE буферного кэша.
SELECT COMPONENT, MIN_SIZE, MAX_SIZE, CURRENT_SIZE FROM V$SGA_DYNAMIC_COMPONENTS
WHERE COMPONENT IN ('KEEP buffer cache', 'RECYCLE buffer cache', 'DEFAULT buffer cache');

8.	Найдите размер свободной памяти в большом пуле.
SELECT POOL, NAME, BYTES FROM V$SGASTAT WHERE POOL = 'large pool' AND NAME = 'free memory';

9.	Получите полный список работающих в настоящее время фоновых процессов.
SELECT SID, PROCESS, NAME, DESCRIPTION, PROGRAM
FROM V$SESSION s JOIN V$BGPROCESS b on s.paddr = b.paddr
WHERE s.STATUS = 'ACTIVE';

10.	Определите режимы текущих соединений с инстансом.
SELECT SID, SERVER, PROGRAM, OSUSER, STATE
FROM V$SESSION WHERE STATUS = 'ACTIVE';
